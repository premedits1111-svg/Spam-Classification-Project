# Spam Message Classification

## Project Overview

This project uses Machine Learning to classify SMS messages as Spam or Ham.

## Dataset

Spam Message Dataset

## Algorithms Used

1. Logistic Regression
2. Random Forest Classifier

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Google Colab

## Project Workflow

1. Data Preprocessing
2. Text Vectorization using TF-IDF
3. Train-Test Split
4. Model Training
5. Cross Validation
6. Performance Evaluation
7. Model Comparison
